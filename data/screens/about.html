About the Interstellar Armada game

Game version: unknown

This application is a learning project in the early development phase. The current version is only intended for testing the initial, already implemented features of what one day might become a functional game. Even the core features are yet under development. The application might contain numerous bugs.

About the used software

This game is created using freely available and/or open source software and open standards. For creating the in-game assets, Inkscape, Blender, Gimp and specialized tools developed (in Netbeans and Lazarus) by the author of the game are used, and the game itself is being developed using Netbeans, Google Chrome and Firefox, all of these running on Ubuntu.
